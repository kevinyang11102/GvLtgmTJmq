# 企业人事管理系统 java214

## 项目概述

企业人事管理系统是一款基于Springboot+Vue的前后端分离的Javaee项目。该系统采用MVC三层架构，结合Spring、SpringBoot、Mybatis和Vue等技术开发，使用MySQL数据库存储数据，实现了浏览器/服务器模式（B/S）的Web应用。其界面清晰，操作便捷。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 项目构建：Maven
- 架构：MVC

## 功能模块

- 用户管理
- 员工信息管理
- 考勤管理
- 薪资管理
- 权限管理
- 报表统计

## 系统角色

- 系统管理员
- 人事专员
- 财务专员
- 员工

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL 5.7+
- JDK版本：1.8+
- Node.js版本：10.0+

## 部署步骤

1. 配置数据库环境
2. 导入数据库脚本
3. 编译后端代码
4. 部署后端服务
5. 前端项目编译
6. 部署前端项目

## 目录结构

```
项目名

```

## 核心亮点

- 前后端分离，便于管理和扩展
- 基于角色的权限控制，保障系统安全
- 界面简洁，操作方便
- 支持多种报表统计与导出
